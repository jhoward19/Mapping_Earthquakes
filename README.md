# Mapping_Earthquakes
In this challenge, three maps were created that displayed earthquakes and tectonic plates. A user can toggler over the overlay and see the options to select or deselect the map options and whether it displays the earthquakes information and tectonic plates. A user can also click on the earthquakes to get detailed information about earthquake like the magnitude and location.
